Project 4
Tic tac toe

A Tic Tac Toe game that allows you to either verse another human or an AI. The AI uses the randint algorithm to determine what space it will choose, so sometimes you will be able to beat the AI but other times it will be impossible! I implemented the play board using the Turtle Python library so you can see where you and your opponent have gone. I also added a little easter egg in AI mode, see if you can find it and good luck beating the AI! Have Fun !
Utilizes: Python, and Turtle
